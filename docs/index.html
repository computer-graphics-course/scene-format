Intro

Top
This scene format is intended to be used to define scenes for rendering as a part of computer graphics course.
We use protobuf to define a files schema (see it on GitHub). Protobuf schema is a single source of truth and it is used to generate io libraries and this document.

Usage & example

Please see the example of demo scene defined in this format. To read or write such files we provide libraries for a number of languages.
It is strongly advised to use libraries we provide instead of implementing your own parser. That is because our libraries implement additional pre/processing steps when reading/writing the file.

C# Library

Step 1. Create a GitHub token with the read:packages scope here.
Step 2. Add a nuget source: 
      
url = "https://github.com/computer-graphics-course/index.json"
user = "<your github username>"
password = "<token from step 1>"
Step 3. Install SceneFormat dependency. Check this page for the latest version available.

Transform

Top

Position, rotation and scale of an object

FieldTypeRequired to implement?Description
position Vector3 yes

rotation Vector3 yes

Rotation in euler angles.
Can be negative or positive.
Angles are measured counterclockwise

scale Vector3 yes

parent_id int32 no


Id of parent relative to which transformation should be applied.
If parent_id is 0 transformation is specified in world space.
To correctly apply transformation you should build a tree structure representing relationships between objects.
Then from top to bottom create transformation matrix.
If object has parent: multiply parent's matrix by object's matrix to get final matrix
